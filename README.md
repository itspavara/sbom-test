# node-js-sample

A simple Node.js application built with Express.js and containerized with Docker.

## Repository Overview

This repository contains a barebones Node.js sample application that demonstrates basic Express server setup and Docker containerization. It includes configuration files for deploying and running the application in various environments.

## Project Structure

- `index.js` - Main application entry point
- `package.json` - Project dependencies and scripts
- `Dockerfile` - Container configuration for deployment
- `app.json` - Application configuration
- `Procfile` - Process file for deployment
- `public/` - Static assets directory
